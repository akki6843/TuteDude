# ASSIGNMENT 6:
## Module 6: Calculator using Tkinter

### Overview:
This project is a simple calculator implemented using Python's `tkinter` library. The application supports basic arithmetic operations such as addition, subtraction, multiplication, and division.

### Features:
- Supports four basic arithmetic operations.
- Displays results with up to two decimal places.
- Handles division by zero gracefully by showing an error message.
- Clear button to reset the calculator.

### Solution: 
The code for this project is available in the `calculator.py` file. You can view and modify it as needed.

- **Calculator Code**: [View on GitHub](https://github.com/akki6843/TuteDude/tree/master/Python/Assignment_06_Tinkter_calculator/calculator.py)

### How to Run:
To run the application, you can clone this repository and navigate to the `calculator.py` file. Then, simply execute it using Python.

```bash
python calculator.py
```
